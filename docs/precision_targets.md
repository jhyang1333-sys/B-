# 论文精度目标对照

本文摘录《氦原子体系能级与极化率的高精度计算》与 Phys. Rev. A 100, 042509 (2019) 中的关键数值，作为后续数值实验的对照标准。所有能量均为原子单位 (a.u.)，极化率为 $a_0^3$。

## 1. 能量谱目标

| 指标              | 参考值               | 允许误差             | 备注                                           |
| ----------------- | -------------------- | -------------------- | ---------------------------------------------- |
| 基态 $1^1S$ 能量  | $-2.903724377034119$ | $\le 1\times10^{-8}$ | 非相对论、含无限核质量修正，来自 PRA 100 表 II |
| 第一激发态 $2^3S$ | $-2.175229378236791$ | $\le 5\times10^{-8}$ | 与 Hylleraas-B-spline 结果一致                 |
| 第一激发态 $2^1S$ | $-2.145974046054417$ | $\le 5\times10^{-8}$ | 需正交化 singlet 通道                          |

> 注：若采用有限核质量 (CODATA 2022) 则能量会抬升 $\sim 3.7\times10^{-4}$ a.u.；本项目默认使用有限核质量设置，验证时需注明。

## 2. Hellmann 判据

| 指标                                             | 目标                 |
| ------------------------------------------------ | -------------------- |
| 基态 Hellmann $\eta = \mid \frac{T+V}{T-V}\mid $ | $\le 1\times10^{-6}$ |
| 低激发态 Hellmann                                | $\le 5\times10^{-6}$ |

数值越接近零表示动能/势能平衡度越高。若 $\eta \gt 10^{-4}$，通常意味着重叠矩阵调节不足或本征向量尚未收敛。

## 3. 静态极化率

| 态          | 长度规范 $\alpha_L$ | 允许误差             | 速度规范一致性 |
| ----------- | ------------------- | -------------------- | -------------- |
| 基态 $1^1S$ | $1.383192174(3)$    | $\le 2\times10^{-6}$ | $              | \eta_{LV} | = \frac{2 | \alpha_L-\alpha_V | }{\alpha_L+\alpha_V} \le 1\times10^{-6}$ |
| $2^3S$      | $315.6315(5)$       | $\le 1\times10^{-4}$ | 同上           |

## 4. 动力学极化率基准

- 频率扫描 $\omega \in [0,0.8]$ a.u.，基态长度/速度/加速度三种规范峰值相对误差 $\le 1\times10^{-3}$。

## 5. 建议流程

1. **小规模验证**：$r_{\max}=20$, $k=5$, $n=5$, $l_{\max}=2$，确认 Hellmann $< 10^{-3}$。
2. **递进式扩大基组**：依次调整 $n$, $r_{\max}$, $l_{\max}$，每一步记录通道裁剪数量与重叠调节方式。
3. **最终大尺度**：当基组维度达到 $10^5$ 级别后改用正则化模式 (`--overlap-conditioning-mode regularize`)，并将容差调低至 $1\times10^{-12}$ 确保不丢失物理解。
